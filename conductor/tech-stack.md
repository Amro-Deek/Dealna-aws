# Technology Stack

This document outlines the technology stack for the Dealna project.

## Backend
- **Language:** Go (Golang)
- **Web Framework/Router:** Chi
- **Database Driver:** pgx for PostgreSQL
- **Authentication:** JSON Web Tokens (JWT)

## Frontend
- **Mobile Application:** Flutter (Dart)
- **Admin Panel:** Next.js (React)

## Database
- **Primary Datastore:** PostgreSQL

## DevOps & Tools
- **Version Control:** Git & GitHub
- **API Testing:** Postman
